# Technology Stack

## Frontend Technologies

### Core
- **React 18**: Modern UI library with concurrent features
- **TypeScript**: Type-safe development
- **Vite**: Next-generation frontend tooling

### UI Framework
- **Shopify Polaris**: Enterprise-grade design system
- **Tailwind CSS**: Utility-first CSS framework
- **Lucide React**: Modern icon library

### State Management & Data Fetching
- **Zustand**: Lightweight state management
- **TanStack Query**: Powerful data synchronization
- **React Router DOM**: Client-side routing

### Data Visualization
- **Recharts**: Composable charting library
- **Chart.js**: Flexible charting options
- **React Leaflet**: Interactive maps

## Backend Services

### Database
- **Supabase**: 
  - PostgreSQL database
  - Real-time subscriptions
  - Row-level security
  - Built-in authentication

### APIs
- **Shopify Admin API**: Store management
- **REST APIs**: Data communication
- **WebSocket**: Real-time updates

## Development Tools

### Code Quality
- **ESLint**: Code linting
- **TypeScript**: Static type checking
- **Prettier**: Code formatting

### Testing
- **Vitest**: Unit testing
- **Testing Library**: Component testing
- **Cypress**: End-to-end testing

### Build & Deploy
- **Vite**: Build system
- **PostCSS**: CSS processing
- **Autoprefixer**: CSS compatibility

### Package Management
- **Yarn**: Fast, reliable dependency management
- **Yarn PnP**: Plug'n'Play module resolution
- **Yarn Workspaces**: Monorepo support
